# Denúncia Falsa (Simulação)

Projeto educativo em React para demonstrar uma página de denúncia falsa. Ao clicar em enviar, o formulário não envia dados; em vez disso, mostra um checklist de verificação institucional.

Estrutura mínima incluída:
- public/index.html
- src/index.js
- src/App.jsx
- src/App.css
- src/components/Checklist.jsx

Como usar:
1. `npx create-react-app my-app` (opcional)
2. Substituir os arquivos em `src/` e `public/` pelos incluídos aqui.
3. `npm install` && `npm start`
